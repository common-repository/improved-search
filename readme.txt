== Description ==

Improved Search is a AJAX search interface you can place anywhere on your WordPress site and get instant search results as soon as you beginning typing. Improved Search delivers the most relevant results first. It also separates results by post type to help you quickly identify the post you are trying to find. Improved search allows you to customize the experience to fit the needs of your WordPress site. Improved Search comes with a shortcode as well as a widget for you to control its location.
